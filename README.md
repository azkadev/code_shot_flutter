### Development

1. Clone source code 

    ```bash
    git clone https://github.com/azkadev/code_shot_flutter.git
    cd code_shot_flutter
    ```

2. Download package

    ```bash
    flutter pub get
    ```

3. Running

    ```bash
    flutter run
    ```
